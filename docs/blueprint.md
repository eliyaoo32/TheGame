# **App Name**: HabitVerse

## Core Features:

- Interactive Habit Dashboard: User-friendly dashboard displaying daily and weekly habit progress, including visualizations and summaries.
- Habit Definition: Allow users to define and customize their habits (daily/weekly, duration, time, boolean, enum).
- Chat-Based Reporting: Receive habit updates via chat/text. A tool extracts data from the text and updates the system.
- AI Habit Feedback: AI-powered feedback system that reminds users of pending habits or congratulates them on completed habits.  The tool intelligently decides if or when a reminder would be helpful.
- Shared Habit View: Ability to see the status of other users' habits within the system (with privacy settings).
- Secure User Accounts: User authentication and authorization.

## Style Guidelines:

- Primary color: Teal (#008080) for growth and balance.
- Background color: Light cyan (#E0FFFF) for a refreshing and clean backdrop.
- Accent color: Coral (#FF7F50) to highlight progress and achievements.
- Body font: 'Roboto' sans-serif for readability and a modern feel.
- Headline font: 'Montserrat' sans-serif for a bold and modern look.
- Use consistent and intuitive icons for each habit type (duration, time, boolean, enum).
- Clean and intuitive layout for the dashboard, focusing on key progress indicators.
